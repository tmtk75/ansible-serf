serf
=========
Ansible role for [serf](http://www.serfdom.io/).
This role ensures that serf runs as a daemon.

Role Variables
--------------
- version
- sha256sum
- consul\_enabled

Dependencies
------------
No dependencies.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: tmtk75.serf }
